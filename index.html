Towards Privacy-Preserving Cloud Storage: A Blockchain Approach

The paper "Towards Privacy-Preserving Cloud Storage: A Blockchain Approach" proposes a data protection scheme called BC-PECK, leveraging blockchain and public key searchable encryption. It encrypts data and stores it in the cloud while establishing a ciphertext index on the blockchain. Smart contracts control data access and sharing, ensuring fairness and security. The scheme addresses privacy concerns in cloud storage and supports multi-keyword queries, enhancing usability. Security analysis confirms confidentiality and fairness, while performance evaluation shows feasibility and efficiency. The scheme combines privacy protection with blockchain's credibility, offering a promising solution for secure cloud storage.
